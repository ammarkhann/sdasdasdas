Our mission is to build an automated depression assessment tool which analyses the user input in different ways. Currently, there are only websites which ask users 5-6 multiple choice questions and then tell them there score. We want to do much more than that.
We want to give the user freedom to asnwer a question in different ways and not just multiple choice questions. Currently we process the user input text and score them. Later on we also want to include features like speech recognition and give users an open choice, how they would like to answer their question.
